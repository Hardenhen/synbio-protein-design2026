# ＡＵＲＯＲＡ GFP Design 2026 — 设计思路文档

## 1. 设计背景与目标

绿色荧光蛋白（GFP）是分子生物学中最广泛使用的报告基因之一。2026 GFP 设计竞赛的目标是：运用计算方法设计兼具**高荧光亮度**和**优良热稳定性**的 GFP 变体。每支队伍提交 6 条序列，经合成、无细胞表达（CFPS）、初始亮度测量、72°C 热处理后亮度测量，综合得分为：

```
单条序列综合得分 = (Finitial / Finitial_WT) × (Ffinal / Finitial)
```

其中若 Finitial < 0.3 × Finitial_WT 则直接淘汰。因此设计时需同时保证：
1. 蛋白能正确折叠并形成功能性生色团（高亮度）
2. 蛋白具有足够的热稳定性抵抗 72°C 高温（高残留活性）

## 2. 种子序列选择理由

### 2.1 为什么不选 sfGFP / avGFP

绝大多数 GFP 工程化研究以 avGFP 或其超折叠变体 sfGFP 为起点。然而本团队刻意选择 3 个与 sfGFP **序列相似度较低**的 GFP 家族成员作为种子，理由如下：

| 参考序列 | PDB | 长度 | 与 sfGFP 相似度 | 来源物种 |
|:--------|:----|:-----|:---------------|:--------|
| amacGFP | 7LG4 | 231 aa | ~70% | *Aequorea macrodactyla* |
| cgreGFP | 2HPW | 231 aa | ~30% | *Clytia gregaria* |
| ppluGFP | 2G6X | 225 aa | ~25% | *Platynereis dumerilii* |

### 2.2 选择低相似度种子的优势

1. **序列多样性最大化**：3 个种子覆盖了 ~25%–70% 的相似度梯度，ProteinMPNN 在不同骨架上生成的序列空间互补，降低所有候选因同源收敛而同时失败的风险
2. **规避排除列表**：与 sfGFP 差异越大，生成的序列越不容易与排除列表中的已知变体冲突
3. **天然热稳定性潜力**：cgreGFP 和 ppluGFP 来自不同物种的天然 GFP，其骨架可能本身已具备不同于 sfGFP 的稳定性特征
4. **生色团保守性**：尽管序列相似度低，GFP 家族的核心生色团三联体（X-Tyr-Gly，对应 sfGFP 的 65-67 位）在所有 3 个种子中保守，确保荧光功能不被破坏

### 2.3 种子序列信息

```
amacGFP (7LG4, 231aa) — 来自刺胞动物，β-barrel 折叠典型
cgreGFP (2HPW, 231aa) — 来自水母 Clytia，序列与 sfGFP 差异显著
ppluGFP (2G6X, 225aa) — 来自多毛类浮游生物，天然短序列接近下限
```

## 4. 算法参数说明

### 4.1 ProteinMPNN 参数

| 参数 | 值 | 说明 |
|:-----|:--|:-----|
| 模型权重 | v_48_020 | ProteinMPNN vanilla 版本 |
| 采样温度 | [0.1, 0.2, 0.5] | 0.1 偏保守（接近原序列），0.5 偏探索 |
| 每温度候选数 | 100 | 每个种子共 300 条候选 |
| 固定残基 | [1, 65, 66, 67, 96, 222] | M1(起始) + 65/66/67(生色团 XYG) + 96(R96催化) + 222(C端) |
| batch_size | 25 | GPU 显存平衡 |
| seed | 42 | 可复现性 |

### 4.2 ESMFold 参数

| 参数 | 值 | 说明 |
|:-----|:--|:-----|
| 模型 | facebook/esmfold_v1 | HuggingFace Transformers |
| num_recycles | 8 | 高回收模式，提升预测精度 |
| chunk_size | 128 | 注意力分块，降低显存峰值 |

### 4.3 评分与筛选

```
sort_score = (pTM × 0.40) + (全局pLDDT / 100 × 0.30) + (生色团pLDDT / 100 × 0.30)
```

- **pTM (0.40)**：全局拓扑正确性，权重最高
- **全局 pLDDT (0.30)**：整体折叠置信度
- **生色团 pLDDT (0.30)**：残基 58-72 区域局部置信度，直接关联荧光功能

## 7. 期望结果与局限性

### 7.1 期望结果

- 900 条候选中预期 50-150 条通过存活门槛（pTM > 0.6）
- Top 6 候选预期能达到 pTM > 0.75、生色团 pLDDT > 65
- 从 3 个不同物种种子出发，最终 Top 6 预期覆盖至少 2 个不同种子来源，增加功能多样性

### 7.2 局限性

1. **ESMFold 精度**：ESMFold 对 GFP 家族的预测精度虽高，但与 AlphaFold3 相比在生色团区域可能存在偏差；若进入最终提交阶段应考虑用更高精度工具复核
2. **ProteinMPNN 单轮设计**：未做多轮迭代（设计→筛选→再设计），仅单轮反向折叠，可能错过需要多步突变才能达到的优质序列
3. **缺乏亮度直接预测**：评分公式仅依赖结构置信度指标，未引入荧光亮度预测模型，存在结构好但亮度低的风险
4. **热稳定性代理指标**：pLDDT 和 pTM 只能间接反映折叠稳定性，无法直接预测 72°C 热处理后的残留活性
5. **固定残基策略**：固定 6 个关键残基可能过度约束，限制了 MPNN 在其他功能位点的探索空间
